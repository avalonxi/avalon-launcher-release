# Avalon Launcher Releases

## Download The Launcher

- Latest release: [github.com/AvalonXI/avalon-launcher-release/releases/latest](https://github.com/AvalonXI/avalon-launcher-release/releases/latest)
- Player support: [discord.gg/avalonxi](https://discord.gg/avalonxi)

## Install And Update

1. Open the latest release page.
2. Download `AvalonXI-Setup.exe`.
3. Run the installer and launch Avalon XI from the Start Menu.
4. Future launcher updates are delivered through the built-in updater.
